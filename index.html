Nightwatch Nedir?

What is Nightwatch?

Nightwatch, Albion Online için geliştirilmiş harici bir ağ paketi okuyucusudur (Packet Sniffer). Oyun dosyalarına veya belleğine müdahale etmez, sadece gelen trafiği analiz eder.

Nightwatch is an external packet sniffer for Albion Online. It analyzes network traffic without modifying game files or RAM memory.

  • Harici Çalışır: Bellek okuma veya kod enjekte etme yapmaz.
  • Hafif: FPS düşürmez, arka planda sessizce çalışır.
  • Purely External: No memory reading or code injection.
  • Lightweight: Zero FPS impact, runs silently.

1. Kaynaklar

1. Resources

  • Kaynakları orijinal ikonlarıyla takip edin.
  • İstemediğiniz türleri tek tıkla gizleyin.
  • Tracker penceresiyle etrafınızdaki yoğunluğu listeleyin.
  • Track resources with original icons.
  • Hide unwanted types with one click.
  • Use the Tracker window to list nearby density.

2. Mobs / Mist

2. Mobs / Mist

  • Boss, Mist ve özel yaratıkları seçerek takip edin.
  • İsimleri göstermeyi veya gizlemeyi seçin.
  • Blacklist ile istemediğiniz yaratıkları karalisteye alın.
  • Track selected Bosses, Mists, and special mobs.
  • Toggle monster names on or off.
  • Use the Blacklist to ignore specific creatures.

3. Player (Oyuncular)

3. Player

  • Oyuncu isimlerini ve Guild bilgilerini görün.
  • Kendi Guild/Alliance üyelerinizi otomatik dost listesine ekleyin.
  • Menzilden çıkanların ekranda kalma süresini (Timeout) belirleyin.
  • See player names and Guild info.
  • Auto-whitelist your own Guild/Alliance members.
  • Set the timeout duration for off-radar players.

4. Config (Yapılandırma)

4. Config

Kendi özel filtre ve radar ayarlarınızı oluşturun, kaydedin ve tek tıkla geri yükleyin.

Create, save, and load your custom filter and radar configurations with a single click.

5. DevTools (Geliştirici)

5. DevTools

  • Simülasyon: Radar görünümünü ve taç ikonlarını test edin.
  • Tracker Yönetimi: Hangi mob/kaynağın takip listesine gireceğini seçin.
  • Sistem: Port (5056/5057) bağlantılarını ve ham konsol loglarını kontrol edin.
  • Parser: Ham oyun verilerini ve nesne özelliklerini detaylıca inceleyin.
  • Simulation: Test radar visuals and crown icons.
  • Tracker Control: Choose which objects hit the tracking list.
  • System: Check ports (5056/5057) and raw console logs.
  • Parser: Inspect raw game data and object properties.

6. Ayarlar (Settings)

6. Settings

  • Arayüz: Radar boyutunu ayarlayın ve Watermark'ı taşıyın (Limit 70 önerilir).
  • Uyarı: Düşman alarmlarını yönetin (Erken uyarı önerilir).
  • OBS Bypass: Radarı yayınlardan gizleyin (Beta aşamasındadır).
  • Kısayollar: Önemli tuşları atayın ve TR/EN dilleri arasında geçiş yapın.
  • UI: Resize the radar and move the Watermark (Draw limit 70 recommended).
  • Alerts: Manage enemy alarms (Early warning recommended).
  • OBS Bypass: Hide radar from streams (In beta).
  • Hotkeys: Bind key controls and switch TR/EN languages.
